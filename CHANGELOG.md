# Changelog

All notable changes to the Test Data Generator Chrome Extension will be documented in this file.

## [1.0.0] - 2026-02-02

### Added
- Initial release with 40+ data generators
- Bilingual support (Arabic/English)
- Saudi-specific data formats and cultural context
- Three access methods: popup, web integration, context menu
- Export functionality (JSON, CSV, clipboard)
- Comprehensive data categories:
  - Personal information (names, IDs, demographics)
  - Contact details (email, phone, address)
  - Work-related data (companies, job titles, salaries)
  - Financial data (IBAN, credit cards, banking)
  - Healthcare information (medical records, prescriptions)
  - Government services (Saudi government data)
  - E-commerce data (products, orders, reviews)
  - Technology data (APIs, databases, software testing)
  - Travel information (hotels, flights, bookings)
  - Education data (students, courses, grades)
  - Real estate information (properties, mortgages)
  - Entertainment data (movies, music, games)
  - Sports data (teams, players, statistics)
  - Food & restaurant data (menus, orders, reviews)
  - Agriculture data (crops, livestock, equipment)
  - Manufacturing data (products, quality control)
  - Testing & QA data (edge cases, performance, security)

### Features
- 100% local data generation (no network requests)
- Privacy-focused design (no data collection)
- Responsive UI with tabbed organization
- Bulk data generation (up to 100 records)
- Smart categorization and filtering
- Persistent user preferences
- Accessibility compliance (WCAG 2.1)
- Cross-browser compatibility

### Security
- No external dependencies
- Minimal required permissions
- Content Security Policy implementation
- No tracking or analytics

### Documentation
- Comprehensive README with usage examples
- Privacy policy documentation
- Chrome Web Store submission checklist
- Build script for production deployment
