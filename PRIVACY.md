# Privacy Policy - Test Data Generator Chrome Extension

**Last Updated: February 2, 2026**

## Overview

Test Data Generator is committed to protecting your privacy. This extension generates test data locally on your device and does not collect, store, or transmit any personal information.

## Data Collection

**We DO NOT collect:**
- Personal information
- Browsing history
- Website data
- Usage analytics
- Generated test data
- Any form of user data

## Data Generation

- All test data is generated locally on your device
- No network requests are made for data generation
- Generated data is not stored or transmitted anywhere
- Data exists only in your browser's memory during use

## Permissions

The extension requests minimal permissions:

- **activeTab**: Required for context menu integration on web pages
- **storage**: Used only to save your preferences locally (e.g., UI settings)
- **contextMenus**: Enables right-click menu options for quick data generation

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

## Data Security

Since no data is collected or transmitted, there are no security risks related to data breaches or unauthorized access to your information.

## Changes to Privacy Policy

Any changes to this privacy policy will be reflected in the extension's documentation and version updates.

## Contact

For privacy-related questions, contact: privacy@testdatagenerator.com
